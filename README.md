Simple app to figure out your Chinese Zodiac.


## Built with
* **HTML5**
* **SASS/CSS3**
* **JavaScript**
**Node JS**
**Express**
**MySQL**


## Installation

You need these to run the program:

* NPM
* Node JS
* Express
* Handlebars

To install the packages: `npm install`
